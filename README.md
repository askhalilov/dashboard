# СтудДоска — студенческая доска объявлений

Рабочий проект на Vue + Vite по теме «Студенческая доска объявлений».

## Что реализовано

- Многостраничный SPA-интерфейс: объявления, учёба, жильё, волонтёрство, сообщество, профиль.
- Интерактивные объявления: просмотр, отклик, избранное, удаление пользовательских объявлений.
- Модальное окно создания объявления.
- Статьи в разделе «Сообщество» и доска идей.
- Анимация загрузки в стиле письма ручкой на листе.
- Простой backend на Node.js без сторонних серверных зависимостей.
- Данные backend хранятся в `server/data/db.json`.

## Backend

Backend добавлен для демонстрации клиент-серверной логики. Он хранит объявления, идеи и обратную связь в JSON-файле.

API:

- `GET /api/health` — проверка сервера.
- `GET /api/ads` — список объявлений.
- `POST /api/ads` — создать объявление.
- `PATCH /api/ads/:id/view` — увеличить просмотры.
- `PATCH /api/ads/:id/respond` — увеличить отклики.
- `DELETE /api/ads/:id` — удалить объявление.
- `GET /api/stats` — статистика.
- `GET /api/ideas` — идеи развития.
- `POST /api/ideas` — добавить идею.
- `POST /api/feedback` — отправить обратную связь.

Если backend не запущен, frontend продолжит работать в локальном режиме через `localStorage`.

## Запуск frontend + backend одной командой

```bash
npm install
npm run dev:full
```

Frontend запустится через Vite, backend — на `http://localhost:3001`.

## Запуск отдельно

Backend:

```bash
npm run backend
```

Frontend:

```bash
npm run dev
```

## Сборка

```bash
npm run build
```

## Основные файлы

- `src/App.vue` — интерфейс и клиентская логика.
- `src/styles.css` — оформление проекта.
- `server/index.js` — backend API.
- `server/data/db.json` — JSON-хранилище.
- `scripts/dev-full.js` — одновременный запуск frontend и backend.
